# watermarking-online-handwriting

MATLAB functions for watermarking online handwriting signals.

This repository provides MATLAB implementations of Least Significant Bit (LSB) and Difference Expansion (DE) watermarking methods for online handwriting signals, together with a demo script for MCYT `.fpg` files.

## Overview

The repository currently includes:

- LSB watermark embedding and extraction
- DE watermark embedding and extraction
- MCYT FPG signature reading
- X/Y centering utilities
- Basic visualization for comparing original, watermarked, and recovered trajectories

## Requirements

- MATLAB R2024b recommended
- Access to MCYT online signature files in `.fpg` format for the demo

No special toolbox is required for the current functions.

## How to run

### 1. Place the repository folders on your computer

For example:

    C:\Users\your_name\Documents\watermarking-online-handwriting\

### 2. Open MATLAB

Open MATLAB and change the current folder to:

    watermarking-online-handwriting/demo

### 3. Edit the input path in the demo

Open:

    demo/demo_mcyt_watermarking.m

and update this line with the location of your MCYT database:

    inputRoot = 'c:\MCYT\firmas\';

### 4. Run the demo

From inside the `demo` folder, run:

    demo_mcyt_watermarking

## What the demo does

The demo script:

1. Reads one MCYT signature in `.fpg` format
2. Centers the X/Y trajectory
3. Embeds a DE watermark into X and Y
4. Extracts the DE watermark and recovers the original signal
5. Embeds LSB watermarks with different numbers of LSBs
6. Extracts the LSB watermarks
7. Displays plots for visual comparison
8. Prints exact extraction checks in the MATLAB command window

## Expected folder behavior

The demo uses:

    addpath('../io', '../de', '../lsb', '../utils');

Therefore, it is intended to be executed **from the `demo` folder**.

If you run the script from another folder, MATLAB may not find the required functions unless you manually adjust the path.

## Main files

### Demo

- `demo/demo_mcyt_watermarking.m`  
  End-to-end example using one MCYT signature file.

### Input/output

- `io/read_fpg_signature.m`  
  Reads MCYT `.fpg` files into MATLAB.

### Difference Expansion

- `de/embed_de_watermark.m`  
  Embeds a binary watermark in X/Y coordinate pairs using DE.
- `de/extract_de_watermark.m`  
  Extracts the DE watermark and recovers the original X/Y signal.
- `de/de_extract_pair.m`  
  Helper function for DE extraction on a single pair.

### LSB watermarking

- `lsb/embed_lsb_watermark.m`  
  Replaces the least significant bits of X and Y.
- `lsb/extract_lsb_watermark.m`  
  Extracts the embedded LSB watermark from X and Y.

### Utilities

- `utils/center_signature_xy.m`  
  Centers the X/Y trajectory around a target location.
- `utils/plot_signature_comparison.m`  
  Displays three trajectory plots for comparison.

## Notes

- LSB watermarking is implemented using **replacement of the `k` least significant bits**, where `k >= 1`.
- DE watermarking is applied to the first two columns of the signal matrix, assumed to correspond to X and Y.
- Signals are processed in memory; no output files are generated by the current demo.
- The demo currently assumes that the input MCYT file is valid and readable with the implemented parser.
